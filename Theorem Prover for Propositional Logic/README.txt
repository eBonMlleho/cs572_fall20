How to run:
Standard Eclipse IDE
Run Main.java file (Main.java contains KB.txt filepath)


About KB.txt file: Variable name and operator format change: 
	Replace all variable strings to single letter in the KB.txt file
	|| operator --> |
	&& operator --> &
	=> operator --> =
	<=> operator --> <
	~ operator --> ~

For example: 
~Rain || ~Outside || Wet		--> ~R | ~O | W
( rain && outside ) => wet		--> (R & O) = W
